Requirements

Skills

  • Being persistent. Have willing to spend many days for figuring out stuff. Also, you have to understand it doesn't always give what you want.
  • Skill to learn and figure out from examples.
  • Problem solving skill.
  • Basic understanding of pointer (like C, C++) and address.
  • Searching skill. You can search from internet for basic MIPS instruction, and you can search old chat in modding server for hint
  • Basic understanding of programming in general. Programming skill also can help you to make some small tools.
  • Basic knowledge about Patapon game modding, especially understanding of resources, like params.

Materials

  • Ghidra
    • Download gzf from here, and *import* to Ghidra (this is not complete, but some functions have name).
  • PPSSPP (latest version if possible) and Patapon 3 game, obviously!
    • Get decrypted eboot.bin for editing. Game will start normally with decrypted Eboot.bin.
      • To dump Eboot.bin, check Game Settings → Tools → Developer tool → Dumped decrypted eboot.bin on game boot. Then start the game, and check SYSTEM/DUMP in PPSSPP memstick folder.
  • Hex editor
    • HxD, ImHex are mainly used hex editors, but some other hex editors are available as editor extension, so grab what you like!
  • Param documents (optional, depending on what you are going to do)
