How To: Create an Enemy Target

How does it work

There is "generator" that contains object definitions.

1. Create a generator with create

This is done in setMissionTimmingScript 1st phase.

Related article: create.

2. Setup generator

This is done in setMissionTimmingScript 1st phase, right after step above

Related instructions: Generator setup

Use 1st parameter from create to define the detail of the generator.

  • setModeEx: Defines hitbox layer (is player side or enemy side)
  • setPosX: Defines position
  • setDeathCall: Defines instruction set to call after the unit is dead.

3. Setup individual unit in the generator.

This is done in setMissionTimmingScript 1st phase, right after step above

  • setReqActorTye:
  • setReqActorCount: How many actors will you define.

4. Spawn the generator

This is done in setMissionTimmingScript 2nd phase.

Related instruction: Spawn

Spawn delay can be set with setReqActorDelayTime.

For structure

  1. Use registConstGimmick (MissionTimming phase 1).
  2. Set the gimmick with applyConstGimmick (MissionTimming phase 2).
  3. Use setGateInfo (last parameter with 1st param 0) to setup the gimmick (registerConstGimmick last parameter).
  4. Control collision status with enableCollisionConstGimmick.

For dungeon

  1. Use setupStartDungeonWall.
  2. Call setupAddDungeonWall.
  3. Set detail information of it with setGateInfo.
  • If there is any lever, create the levers first and get thd ID of them.
  1. Finish with setupEndDungeonWall

Spawning units from a structure

  1. Create the tower using registConstGimmick.
  2. Use generator and connect with SetModeEx last parameter (to registeConstGimmick last parameter).
  3. Set the time between respawn and total respawn count.
  4. Use spawn to spawn units.
