setupUnit

Loads unit settings, e.g. character, equipment information, and skills. applyUnit after this is required to apply this setup.

Code Information

  • Name: setupUnit
  • Scope: Global
  • PAC Instruction (Binary): 25 16 33 00
  • Assembly Address in Memory : 0x891cc84

Parameters

This function is parameterless.

25 16 33 00
