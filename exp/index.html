Exp Calculation

How to calculate

The Code 
   /* max_lvlgap_for_exp is 2 */
    int Calculate_Exp(float base_exp,int player_level,int enemy_level)

    {
      int final_exp;
      int *final_leveldiff;
      int level_diff;
      
      level_diff = enemy_level - player_level;
      final_exp = 0;
      if (-3 < level_diff) {
        if (level_diff < max_lvlgap_for_exp) {
          final_leveldiff = &level_diff;
        }
        else {
          final_leveldiff = &max_lvlgap_for_exp;
        }
        final_exp = (int)(base_exp + base_exp * (float)*final_leveldiff * 0.35);
      }
      return final_exp;
    }

Which means Base_Exp + Base_Exp * min(enemy_level - player_level, 2) * 0.35

  • If your level is 3 or higher than enemy, you cannot get exp.
  • If your level is 2 or more lower than enemy, you get same max exp.
